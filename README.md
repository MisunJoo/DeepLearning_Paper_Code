# DeepLearning Paper with Code 
## 스터디 개요
-   스터디 목표
    -   주요 딥러닝 논문을 보며 모델링을 할 때 어떻게 코드로 작성할 수 있는지를 학습
-   스터디 방법
    -   **매주 금요일**에 모여 돌아가면서 그 주의 논문을 읽고 코드로 구현하고 와서 각자 맡은 파트를 스터디 원에게 설명
